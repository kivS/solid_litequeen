The main application, the gem, is located at the root of the application and it's a gem for managing Sqlite databases for Rails projects.
We also have a nextjs documentation website(our marketing site) living in `website/`

## Project Goal

An engine we can mount to manage SQLITE databases in the Rails app.


## Tech highlight for the project

- Ruby on Rails engine
    - https://guides.rubyonrails.org/engines.html



- Turbo for frontend
    - https://turbo.hotwired.dev/handbook/introduction

    - https://turbo.hotwired.dev/reference/drive

    - https://github.com/hotwired/turbo
    

- Kamal to deploy the marketing site to our VPS
    - https://kamal-deploy.org/docs/configuration/overview/
    - .kamal/
    - config/deploy.yml


- StimulusJS for frontend
    - https://stimulus.hotwired.dev/handbook/introduction
    - https://stimulus.hotwired.dev/reference/controllers
    - https://github.com/hotwired/stimulus


- Tailwindcss v4 for styling
    - https://tailwindcss.com/docs
    - https://github.com/rails/tailwindcss-rails
    - config located in `app/assets/tailwind/application.css`
